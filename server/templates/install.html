Install in 3 steps

  1. Download the agent and install dependencies: 
    git clone <your-reachly-repo> reachly
cd reachly
pip install -r requirements.txt
python -m playwright install chromium   # only if you use browser mode
  2. Copy .env.example to .env and fill in your business info, your AI keys, and your platform credentials.
  3. Preview, then go live: 
    python -m reachly.runner preview     # writes a post, shows it, never posts
# set DRY_RUN="no" in .env, then:
python -m reachly.runner run         # posts daily at POST_TIME

Tip: run it under systemd, tmux, or Docker so it keeps running. A sample Dockerfile and unit file are in deploy/.
